nRootTag 攻击曝光：苹果 Find My 成“帮凶”，成功率 90%、分钟级追踪几乎任意设备

Thu, 27 Feb 2025 03:23:34 GMT
<p>IT之家 2 月 27 日消息，乔治梅森大学本月发布博文，发现名为“nRootTag”的攻击方式，<strong>结合设备的蓝牙地址和苹果 Find My 网络，可以实现追踪几乎所有电脑或移动设备位置。</strong></p><p>IT之家援引博文介绍，“nRootTag”攻击核心原理是诱骗苹果的 Find My 网络，让其误认为目标设备是一个丢失的 AirTag。</p><p>AirTag 会向附近的苹果设备发送蓝牙消息，这些设备再通过苹果云匿名地将 AirTag 的位置传递给其所有者进行追踪。</p><p>该攻击方法可以在未经苹果许可的情况下，将台式机、智能手机或物联网设备等变成“AirTag”，让 Find My 网络开始追踪这些设备。</p><p>研究人员能够精确定位静止电脑的位置（误差在 10 英尺以内），准确追踪移动电动自行车的城市路线，甚至重建游戏机在飞机上的确切飞行路径并识别航班号。</p><p>虽然 AirTag 的蓝牙地址会根据加密密钥进行更改，但攻击者在没有管理员权限的情况下无法在其他系统上进行此操作。</p><p>因此，研究人员没有尝试修改蓝牙地址，而是开发了高效的密钥搜索技术，找到与蓝牙地址兼容的密钥，使密钥适应地址。</p><p>nRootTag 攻击的成功率高达 90%，并且能够在几分钟内追踪设备。该技术不需要通常进行此类深度系统访问所需的高级管理员权限提升，而是巧妙地操纵了 Find My 网络对设备信号的信任，将苹果的 Find My 功能变成了不知情的帮凶。</p>

微软 Windows 365 Link 真机照出炉：仅用于连接 Win11 云电脑，4 月上市

Thu, 27 Feb 2025 03:13:06 GMT
<p>IT之家 2 月 27 日消息，去年，微软推出了一款与众不同的 Windows 桌面迷你电脑 ——Windows 365 Link。这款设备并非传统意义上的迷你电脑，而是一款瘦客户端（thin client），专为通过云端串流 Windows 11 系统而设计。在本地，<strong>其运行的是一个极其轻量级的 Windows 版本，其唯一功能是将用户连接到雇主在云端设置的 Windows 365 PC。</strong></p><p style="text-align: center;"><img class="no-alt-img" src="https://img.ithome.com/newsuploadfiles/2025/2/1e3d9fb2-9b2f-44ab-a8c0-2b2c457faf6c.png?x-bce-process=image/format,f_auto" /></p><p>Windows 365 Link 是微软逐步引入的一种新型 Windows 电脑的开端。<strong>这类电脑并不直接运行 Windows 应用程序</strong>，而是通过互联网和远程桌面协议将 Windows 体验流式传输给用户。</p><p>这种设计带来了诸多优势，<strong>其中最重要的就是安全性</strong>。由于 Windows 365 电脑处理的数据从不存储在本地，因此其安全性更高。即使设备被盗，所有数据依然安全地存储在 Windows 365 服务的云端电脑中。</p><p>此外，<strong>Windows 365 Link 还有助于减少电子垃圾</strong>。由于这类瘦客户端设备不依赖内部硬件规格来来实现其功能，因此即使很长时间之后，也不太会因为内部规格落后而被淘汰丢弃。用户可以通过升级云端电脑的 CPU、内存和存储空间来满足需求，而无需更换设备。</p><p>现在 Windows 365 Link 的实物图片已经曝光，这表明该设备即将面向大众推出。目前，该设备正在与部分 Windows 365 客户进行内部测试，预计将于今年 4 月正式上市。</p><p style="text-align: center;"><img class="no-alt-img" src="https://img.ithome.com/newsuploadfiles/2025/2/e4c94208-23b0-4061-bc53-cd51efe3c8f2.jpg?x-bce-process=image/format,f_auto" /></p><p>从图片来看，Windows 365 Link 体积小巧，单手即可握持。其外壳采用塑料材质，设备正面配备了一个 USB-A 接口和一个 3.5 毫米音频插孔，背面则有另外两个 USB-A 接口、一个 USB-C 接口、以太网接口、DisplayPort 和 HDMI 接口。</p><p>在硬件配置方面，Windows 365 Link 搭载了一颗低端的英特尔 N250 处理器、8GB 内存和 64GB 存储空间。在普通电脑上，这样的配置显得有些寒酸，但由于该设备本身并不运行任何应用程序，用户并不会察觉到性能不足。</p><p style="text-align: center;"><img class="no-alt-img" src="https://img.ithome.com/newsuploadfiles/2025/2/612f492b-044e-4ff4-90fc-d886ee37c547.jpg?x-bce-process=image/format,f_auto" /></p><p>这种低端配置也使得微软能够以更具竞争力的价格推出该设备。<strong>Windows 365 Link 预计将于 4 月以 349 美元（IT之家备注：当前约 2534 元人民币）的价格上市</strong>。</p><p>当然，<strong>这也意味着 Windows 365 Link 离不开 Windows 365 订阅服务</strong>。目前，Windows 365 仅为企业服务，因此用户只有在工作场所获得分配时才可能接触到 Windows 365 Link。</p><p>不过，微软未来将 Windows 365 推向消费市场的可能性不言而喻。事实上，微软已经明确表示有此计划。尽管目前尚不清楚这一服务何时会向消费者开放，但可以推测或许并不遥远。</p>

SpaceX 火箭携私人月球着陆器、NASA 轨道器升空，开启月球寻水之旅

Thu, 27 Feb 2025 02:55:22 GMT
<p>IT之家 2 月 27 日消息，北京时间今早，SpaceX 的猎鹰 9 号火箭从美国国家航空航天局（NASA）肯尼迪航天中心（KSC）发射升空，将 Athena 私人探测器和 NASA 的月球先锋（Lunar Trailblazer）轨道器送入太空。</p><p style="text-align: center;"><img class="no-alt-img" src="https://img.ithome.com/newsuploadfiles/2025/2/7b204896-15e2-471f-b5cf-0f4439a162ab.png?x-bce-process=image/format,f_auto" /></p><p>Athena 探测器由总部位于休斯顿的太空飞行公司 Intuitive Machines 制造，搭载了 10 台 NASA 科学仪器，<strong>其中许多旨在寻找月球水冰的迹象</strong>，月球先锋轨道器则将在更高的轨道上开展类似工作。这些任务对 NASA 至关重要，因为该机构计划通过其阿尔忒弥斯计划在月球上建立一个人类居住点。</p><p>NASA 科学任务局副主任 Nicky Fox 在发射前的新闻发布会上表示：“我们对技术演示所交付的科学成果感到非常兴奋，这将为人类重返月球以及前往火星的旅程做好准备。”</p><p>猎鹰 9 号火箭点火升空大约 8.5 分钟后，火箭的第一级助推器成功返回地球，并在 SpaceX 的“引力不足”号无人船上按计划着陆。这是猎鹰 9 号助推器（B1083）的第九次发射和着陆。</p><p>火箭的上级继续飞行，于发射后 43.5 分钟将 Athena 送入月球转移轨道，随后在四分钟后按计划部署了月球先锋轨道器。</p><p><strong>Athena 的任务被称为 IM-2</strong>，因为这是 Intuitive Machines 在过去两年内第二次向月球发射探测器。IM-2 是 NASA 商业月球有效载荷服务（CLPS）计划的一部分，该计划旨在通过商业合作将科学和技术设备送往月球。</p><p>Athena 的雄心勃勃的任务集中在月球南极的水冰和其他资源上，并探索未来任务利用这些资源实现可持续太空居住的可能性 —— 这一过程被称为原位资源利用（ISRU）。为了实现 IM-2 的所有目标，探测器携带了两辆辅助车辆：一辆名为 MAPP（移动自主勘探平台）的迷你探测车，由科罗拉多州的 Lunar Outpost 公司制造；以及一个名为 Grace 的 Intuitive Machines“跳跃”机器人，其将通过跳跃探索着陆点周围的区域。</p><p style="text-align: center;"><img class="no-alt-img" src="https://img.ithome.com/newsuploadfiles/2025/2/3edd4778-7ebe-4d25-ad96-1fab964786a0.jpg?x-bce-process=image/format,f_auto" /></p><p>IM-2 任务是在 Intuitive Machines 的历史性首次任务 IM-1 之后仅一年多的时间内发射的。IM-1 探测器“奥德修斯”于 2024 年 2 月发射，也是 CLPS 任务的一部分。<strong>奥德修斯成功着陆月球表面，成为首个实现软着陆的私人航天器</strong>。然而，由于着陆速度过快，探测器的一条着陆腿折断，导致其在着陆时翻倒，进而阻挡了高增益天线，使得 NASA 未能收集到部分预期数据。</p><p>“这次，我们希望能够更精准地着陆。”Intuitive Machines 太空系统高级副总裁 Trent Martin 在发射前对 <span class="link-text-start-with-http">Space.com</span> 表示。</p><p>Athena 将前往月球的蒙斯蒙通地区，科学家们认为那里有足够的水冰沉积物可供 ISRU 研究。如果一切顺利，<strong>探测器将在四到五天后进入<strong>月</strong>球轨道，随后在 1.5 到三天后着陆，并在月球表面运行约 10 个地球日。</strong></p><p>探测器的主要科学设备被称为 PRIME-1（极地资源冰开采实验 1），包括两个独立的仪器：用于探索新地形的表岩屑冰钻（TRIDENT）和用于观察月球操作的质谱仪（MSolo）。这对仪器将协同工作，收集并分析从月球表面以下回收的样本。TRIDENT 可以从高达 1 米的深度提取材料，而 MSolo 将测试样本中的化合物，如水和二氧化碳。</p><p>IM-2 的“跳跃”机器人 —— 以开创性计算机科学家和数学家格蕾丝・霍珀命名的 Grace，被设计为在 Athena 着陆区附近约 1.6 公里范围内进行探索。Grace 将利用其推进器、惯性测量单元、星体跟踪器、激光雷达和态势感知相机从一个位置跳到另一个位置。<strong>其主要目标之一是附近陨石坑的永久阴影区 —— 这种环境是轮式探测车无法探索的。</strong></p><p>陨石坑底部将超出 Grace 与 Athena 通信所需的视线范围。MAPP 探测车将填补这一通信空白，<strong>因为其配备了月球首个蜂窝网络</strong>。MAPP 将在月球表面部署，测试月球表面通信系统（LSCS）的高速、长距离通信能力。SCLS 由诺基亚贝尔实验室开发，使用现有 4G / LTE 网络中已有的组件。</p><p>Athena 还配备了一个激光反射器阵列（LRA）—— 八个固定在着陆器上的镜子，用于测试激光光线的反射，以便为轨道和接近的航天器提供更精确的定位。这一概念类似于帮助引导飞机降落的跑道反射器，且 LRA 无需电源或机械控制即可运行。</p><p>与此同时，<strong>月球先锋轨道器将进入月球轨道，开始绘制月球表面的水冰沉积物地图</strong>。NASA 表示，该探测器的数据将与 Athena 仪器收集的近距离信息相辅相成，尤其是 PRIME-1 任务的数据。</p><p>“这将使我们能够将 PRIME-1 任务的发现进行情境化，并扩展到月球其他可能的区域。”月球先锋轨道器首席研究员 Bethany Ehlmann 在发射前表示。</p><p>NASA 最初以 4700 万美元（IT之家备注：当前约 3.41 亿元人民币）购买了 IM-2
