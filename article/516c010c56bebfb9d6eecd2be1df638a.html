微软 Windows Terminal 现免费接入 GitHub Copilot：每月 50 条，AI 命令行助手

分类: 科技 发布时间: Thu, 27 Feb 2025 05:33:14 GMT 来源: 原文链接
IT之家 2 月 27 日消息，科技媒体 WinBuzz 昨日（2 月 26 日）发布博文，报道称微软旗下的 Windows Terminal 现已免费接入  GitHub Copilot AI 助手，为开发者提供命令行支持。免费用户每月可以使用 50 条 AI 聊天消息，而付费用户则不限次数。Windows Terminal 用户可以请求解释命令语法、排除错误以及在不同 shell 环境之间转换命令，AI 助手还具备上下文感知响应，会根据终端会话提供定制响应。Windows Terminal 是微软开发的一款现代化、快速、高效、功能强大的终端应用程序，适用于命令行工具和 shell 的用户。IT之家援引博文介绍，简要回顾下发展历程：2024 年 10 月：Windows Terminal 首次引入 Copilot，但仅限付费用户使用。2024 年 12 月：GitHub 推出 AI 驱动的代码审查和代码引用功能。2025 年 2 月：Visual Studio Code 新增“下一个编辑建议”功能，并推出 Gemini Code Assist Free，成为 Copilot 的强劲竞争对手。目前：Windows Terminal 的 Canary 测试版已向免费用户开放 Copilot 的聊天功能。
