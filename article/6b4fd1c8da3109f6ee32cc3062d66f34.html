微软 Windows 365 Link 真机照出炉：仅用于连接 Win11 云电脑，4 月上市

分类: 科技 发布时间: Thu, 27 Feb 2025 03:13:06 GMT 来源: 原文链接

IT之家 2 月 27 日消息，去年，微软推出了一款与众不同的 Windows 桌面迷你电脑 ——Windows 365 Link。这款设备并非传统意义上的迷你电脑，而是一款瘦客户端（thin client），专为通过云端串流 Windows 11 系统而设计。在本地，其运行的是一个极其轻量级的 Windows 版本，其唯一功能是将用户连接到雇主在云端设置的 Windows 365 PC。

Windows 365 Link 是微软逐步引入的一种新型 Windows 电脑的开端。这类电脑并不直接运行 Windows 应用程序，而是通过互联网和远程桌面协议将 Windows 体验流式传输给用户。

这种设计带来了诸多优势，其中最重要的就是安全性。由于 Windows 365 电脑处理的数据从不存储在本地，因此其安全性更高。即使设备被盗，所有数据依然安全地存储在 Windows 365 服务的云端电脑中。

此外，Windows 365 Link 还有助于减少电子垃圾。由于这类瘦客户端设备不依赖内部硬件规格来来实现其功能，因此即使很长时间之后，也不太会因为内部规格落后而被淘汰丢弃。用户可以通过升级云端电脑的 CPU、内存和存储空间来满足需求，而无需更换设备。

现在 Windows 365 Link 的实物图片已经曝光，这表明该设备即将面向大众推出。目前，该设备正在与部分 Windows 365 客户进行内部测试，预计将于今年 4 月正式上市。

从图片来看，Windows 365 Link 体积小巧，单手即可握持。其外壳采用塑料材质，设备正面配备了一个 USB-A 接口和一个 3.5 毫米音频插孔，背面则有另外两个 USB-A 接口、一个 USB-C 接口、以太网接口、DisplayPort 和 HDMI 接口。

在硬件配置方面，Windows 365 Link 搭载了一颗低端的英特尔 N250 处理器、8GB 内存和 64GB 存储空间。在普通电脑上，这样的配置显得有些寒酸，但由于该设备本身并不运行任何应用程序，用户并不会察觉到性能不足。

这种低端配置也使得微软能够以更具竞争力的价格推出该设备。Windows 365 Link 预计将于 4 月以 349 美元（IT之家备注：当前约 2534 元人民币）的价格上市。

当然，这也意味着 Windows 365 Link 离不开 Windows 365 订阅服务。目前，Windows 365 仅为企业服务，因此用户只有在工作场所获得分配时才可能接触到 Windows 365 Link。

不过，微软未来将 Windows 365 推向消费市场的可能性不言而喻。事实上，微软已经明确表示有此计划。尽管目前尚不清楚这一服务何时会向消费者开放，但可以推测或许并不遥远。
