Anthropic 全面开放 Claude AI 的 GitHub 集成，赋能所有开发者

分类: 科技 发布时间: Thu, 27 Feb 2025 03:37:20 GMT 来源: 原文链接

IT之家 2 月 27 日消息，Anthropic 的 Claude 关系负责人 Alex Albert 于 2 月 25 日在 X 平台发布推文，宣布面向免费、Pro 和 Teams 用户，开放 GitHub 集成 Claude。

GitHub 集成此前仅限 Claude 企业版（Claude for Enterprise），为企业用户提供更强大的代码理解和辅助能力。

集成方式主要为将 GitHub 代码库直接同步到 Claude，让 Claude 能够理解代码库的上下文，帮助用户测试和调试代码、理解代码库并提供更有效的代码辅助功能等。

需要注意的是，免费账户使用此功能可能会消耗大量配额，而 Pro 用户的配额消耗则相对更容易控制。继早前与 Google Drive 集成之后，GitHub 集成是 Claude 界面为开发者提供的第二个重要功能拓展。

除了 GitHub，Anthropic 还在积极开发集成 Google Calendar、Gmail、Salesforce 和 Slack 等服务。
